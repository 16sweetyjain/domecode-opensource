What is Fusion?


Fusion is a real-time disposable editor using which you can start learning front-end technology including HTML, CSS, JS with any YouTube tutorial of your choice in a single tab of a code editor and two browser windows like a memory hog.



How to use it?


  1. Pick any front-end tutorial of your choice that includes HTML, CSS/JS. DomeCode recommends this.
  2. Copy the embed code. You can get the embed code by clicking on the "Share" button under the video.
  3. Paste the embed code to Fusion's Immerse. Make sure to set the height and width attributes accordingly.
  4. Voila! Watch your tutorial, code and even get real-time preview in a single tab!
